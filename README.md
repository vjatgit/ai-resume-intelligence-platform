# AI Resume Intelligence Platform

An AI-powered application that compares a resume with a job description and provides explainable job-alignment insights.

## Planned Features

- Upload PDF and DOCX resumes
- Extract resume content
- Identify matching and missing skills
- Calculate a transparent compatibility score
- Generate resume improvement suggestions
- Create an improved professional summary
- Download an analysis report

## Technology Stack

- Python 3.11
- Streamlit
- PyMuPDF
- python-docx
- Pandas

## Project Status

Initial project setup is complete. Resume document processing is the next development milestone.
